# AllJim

An All-in-one streamlined gym management platform desgined to unify member onboarding, class scheduling, and business operations into a single, high-performance interface.

## Tech Stack

* **[Next.js](https://nextjs.org)** – Full-stack React framework

* **[Vitest](https://vitest.dev)** – Unit testing
* **[Lucide](https://lucide.dev)** – Icon library
